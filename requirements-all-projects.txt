# ============================================================
# requirements-all-projects.txt
#   Consolidated Python dependencies for ALL Huni projects.
#   Sources: SimulGenVAE, SimulGenVAE2D, PEMTRON_warpage,
#            temp/LLM_API, LLM_API_fast sub-bots, Gmail_exporter.
#
#   Packages already handled in dedicated wheel dirs
#   (llm_api, meshgraphnets, vllm, jupyter, llamacpp) are
#   intentionally kept here too so each venv is self-contained.
#
#   Excluded:
#     - pyreadline3      (Windows-only)
#     - langchain-classic (not on PyPI)
#     - xlwt             (Python 2, deprecated)
#     - aider-chat       (dev tool, huge, not needed on server)
#     - llama-cpp-python (needs CUDA compilation; handled via source)
#     - pyinstaller      (build tool, not needed on server)
# ============================================================
